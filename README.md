# Java JIT Demonstration

This project demonstrates how Java handles repeated code execution using JVM and JIT compilation.

## Concepts Covered
- Bytecode execution
- Interpreter vs JIT
- Hot code optimization

## How it works
A large loop is executed multiple times. The JVM identifies this as hot code and optimizes it using JIT, improving performance.

## How to Run
1. Compile:
   javac JITDemo.java

2. Run:
   java JITDemo
